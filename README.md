# nickspaargaren.nl

Personal portfolio website, built with Gatsby, Sanity CMS, and TypeScript as a Yarn monorepo.

## Tech stack

- **Frontend:** Gatsby, React, TypeScript, Styled Components
- **Backend:** Sanity CMS
- **Tooling:** Turborepo, Yarn, Cypress, Vercel

## Development

Run `yarn install` and `yarn dev`, the frontend will be available on `http://localhost:8000` and the backend on `http://localhost:3333`.

## Requirements

- Nodejs
- Yarn
